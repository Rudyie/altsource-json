DelistStore
is a WIP AltStore/SideStore source intended to restore and preserve older delisted iOS AppStore pages, alongside some iOS homebrew utilities

3 main branches of this repo exist to allow testing easily, so allow me to explain each's use: 

1st is the main branch, which has two test apps and is older due to being a stable version, but is up without interuptions.

2nd is test, which acts as a beta version (currently is the same as main) but the json source may break with updates.

3rd is the alpha branch, which adds the DelistStore branding and is used as a (very unfinished) test branch
